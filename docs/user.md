# User

Before an new user is setup you'll need to choise how much you want to integrate this user in nixos.

Below is a list of options nixos (in this flake) can manage for the user

- Password managed by sops
- Other secrets
- Home manager

To setup a new user there are a few things you need to do:

1. [Prep user](./prep/user.md)
2. [Install user on device](./instalation/user.md)
